# New Home Design&Management - Vue3 项目

这是一个基于Vue3的现代化设计工作室网站，具有响应式设计、多语言支持和丰富的动画效果。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 <http://localhost:5173> 查看项目

### 构建生产版本

```bash
npm run build
```

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 快速的前端构建工具
- **Vue Router** - 官方路由管理器
- **Pinia** - 状态管理
- **Vue I18n** - 国际化支持
- **UnoCSS** - 原子化CSS框架
- **VueUse** - Vue组合式工具库

## 功能特性

### 🎨 设计特性

- 现代化的UI设计
- 响应式布局，支持各种设备
- 丰富的动画效果和交互
- 自定义主题色彩系统

### 🌍 多语言支持

- 中英文切换
- 基于Vue I18n的国际化方案
- 语言偏好本地存储

### 📱 页面功能

- **首页**: Hero区域、服务介绍、CTA区域
- **关于我们**: 公司介绍和团队信息
- **服务项目**: 详细的服务介绍
- **项目案例**: 项目展示和分类筛选
- **联系我们**: 联系表单和联系信息

## 项目结构

```
src/
├── components/          # 组件
│   ├── layout/         # 布局组件
│   └── sections/       # 页面区块组件
├── views/              # 页面组件
├── composables/        # 组合式函数
├── stores/             # Pinia状态管理
├── locales/            # 国际化语言包
├── router/             # 路由配置
└── assets/             # 静态资源
```
